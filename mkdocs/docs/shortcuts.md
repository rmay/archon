# Shortcuts

## xbindkeys
### Arhcon terminal
"archon-terminal"  
Mod4 + Return

### Archon dmenu launcher
"archon_dmenu_run"  
  Mod4 + grave

### Archon rofi launcher
"rofi -show run"  
  Mod4 + space

### X terminal
"xterm"  
  Shift + Mod4 + Return

### PrintScreen → region screenshot
"flameshot gui"  
    Alt + F10

### Shift+PrintScreen → full screen
"flameshot full -p ~/Pictures"  
    Alt + F11

### Ctrl+PrintScreen → active window
"flameshot screen -p ~/Pictures"  
    Alt + F12
